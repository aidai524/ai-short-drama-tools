# 镜头语言词典 / Cinema Glossary

分镜拆解时使用的标准镜头语言术语中英对照参考。

---

## 景别 / Shot Types

| 中文 | 英文 | 缩写 | 说明 |
|------|------|------|------|
| 大特写 | Extreme Close-Up | ECU | 只展示五官某部分（眼睛、嘴唇）或物体细节 |
| 特写 | Close-Up | CU | 面部或物体完整占据画面 |
| 近景 | Medium Close-Up | MCU | 胸部以上，清晰展示面部表情 |
| 中景 | Medium Shot | MS | 腰部以上，展示上半身动作和表情 |
| 中远景 | Medium Long Shot | MLS | 膝盖以上，展示大半身和部分环境 |
| 远景 | Wide Shot / Long Shot | WS | 完整人物全身 + 周围环境 |
| 大远景 | Extreme Wide Shot | EWS | 人物很小，主要展示宏大环境 |
| 过肩镜头 | Over-the-Shoulder | OTS | 从一人肩后拍另一人，对话场景常用 |
| 主观镜头 | POV / First-Person | POV | 模拟角色视角 |
| 鸟瞰 | Bird's Eye / Top-Down | BIRDS | 正上方俯拍，上帝视角 |
| 低角度 | Low Angle | LOW | 从低处仰拍，角色显得高大/强势 |
| 高角度 | High Angle | HIGH | 从高处俯拍，角色显得渺小/弱势 |
| 荷兰角 | Dutch Angle / Tilted | DUTCH | 画面倾斜，营造不安/失衡感 |
| 全景 | Full Shot | FS | 人物全身完整可见，适合动作展示 |

### 景别选择指南

| 叙事目的 | 推荐景别 | 原因 |
|----------|----------|------|
| 建立环境 | WS / EWS | 展示空间关系 |
| 角色登场 | MS → CU | 先展示身份，再聚焦面容 |
| 情感表达 | CU / ECU | 捕捉微表情 |
| 对话场景 | MCU / OTS | 交替正反打 |
| 动作场景 | FS / MS | 展示完整动作 |
| 暗示权力关系 | LOW / HIGH | 低角度=强势，高角度=弱势 |
| 内心独白 | CU / MCU | 聚焦面部变化 |
| 群戏 | WS / MS | 展示多人关系 |

## 运镜 / Camera Movements

| 中文 | 英文 | 说明 | 情绪效果 |
|------|------|------|----------|
| 固定机位 | Static / Fixed | 摄影机不动 | 稳重、客观、凝视感 |
| 推近 | Push-In / Dolly In | 镜头向主体靠近 | 强调、聚焦、逼近真相 |
| 拉远 | Pull-Out / Dolly Out | 镜头远离主体 | 孤独感、揭示环境、疏离 |
| 平移 | Pan Left/Right | 水平旋转 | 扫视环境、跟随视线、发现新元素 |
| 俯仰 | Tilt Up/Down | 垂直旋转 | 展示高度、仰视/俯视切换 |
| 跟拍 | Tracking Shot | 跟随移动主体 | 代入感、紧迫感、伴随感 |
| 环绕 | Orbit / Arc Shot | 围绕主体旋转 | 英雄时刻、展示全貌、情绪升级 |
| 升降 | Crane Up/Down | 垂直升降运动 | 宏大感、上帝视角、情绪释放 |
| 手持 | Handheld | 轻微晃动 | 纪实感、紧张不安、临场感 |
| 稳定器 | Stabilized / Gimbal | 流畅移动 | 精致、专业、优雅 |
| 快切 | Whip Pan / Fast Cut | 快速水平甩镜头 | 转场、震惊、时间跳跃 |
| 急推 | Fast Push-In | 快速推近主体 | 震惊、发现关键线索、高潮爆发 |
| 渐亮 | Fade In | 从黑屏/白屏渐显 | 新开始、时间流逝、重生 |
| 渐暗 | Fade Out | 画面渐变黑屏/白屏 | 结束、死亡、时间跳跃 |
| 跟焦 | Follow Focus | 焦点在主体间切换 | 引导注意力、暗示关系 |
| 抓拍 | Grab Shot | 快速捕捉瞬间 | 真实感、突发性、不可预测 |
| 闪回 | Flashback | 快速切换到过去 | 回忆、对比、因果揭示 |
| 慢动作 | Slow Motion | 降速播放 | 强调关键时刻、诗意感、悲剧感 |
| 缩放 | Zoom In/Out | 镜头焦距变化（非移动） | 压缩/扩展空间、窥视感 |
| 主观移动 | POV Movement | 以角色视角移动 | 沉浸感、恐惧探索 |

### 运镜选择指南

| 叙事目的 | 推荐运镜 | 原因 |
|----------|----------|------|
| 偷窥/隐藏视角 | 手持 + 缓慢平移 | 纪实感、不稳定性 |
| 真相揭示 | 急推 → 固定 | 震惊 → 凝视 |
| 角色觉醒 | 渐亮 → 缓慢推近 | 从模糊到清晰 |
| 追逐/逃跑 | 跟拍 + 手持 | 紧迫感、混乱感 |
| 对峙/冲突 | 固定 → 快切 | 紧张对峙 → 爆发 |
| 孤独/绝望 | 缓慢拉远 | 越来越渺小 |
| 英雄时刻 | 低角度 + 环绕 | 力量感、仪式感 |
| 日常场景 | 稳定器跟拍 | 自然流畅 |
| 转场 | 渐暗 → 渐亮 / 快切 | 场景切换 |

## 光影 / Lighting

| 中文 | 英文 | 说明 | 情绪效果 |
|------|------|------|----------|
| 自然光 | Natural Lighting | 模拟日光/月光 | 真实、朴素、日常 |
| 逆光 | Backlit / Rim Lighting | 光源在主体后方 | 剪影、神秘、神圣感 |
| 侧光 | Side Lighting | 从侧面打光 | 立体感、戏剧性、阴阳分割 |
| 顶光 | Overhead Lighting | 从正上方打光 | 审讯感、压迫、疏离 |
| 柔光 | Soft / Diffused | 均匀散射光 | 温柔、梦幻、安全 |
| 硬光 | Hard / Direct | 强烈直射光 | 强硬、对比鲜明、冲突感 |
| 暖光 | Warm / Golden Hour | 黄/橙色调光 | 怀旧、温暖、亲密 |
| 冷光 | Cool / Blue Tone | 蓝/灰色调光 | 疏离、冷酷、现代 |
| 霓虹光 | Neon | 彩色霓虹灯 | 赛博朋克、夜生活、迷幻 |
| 烛光 | Candlelight | 温暖微弱的光 | 亲密、古典、秘密 |
| 明暗对比 | Chiaroscuro | 强烈的明暗交替 | 悬疑、冲突、道德灰色地带 |
| 底光 | Under Lighting | 从下方打光 | 恐怖、诡异、非自然 |

## 转场 / Transitions

| 中文 | 英文 | 说明 |
|------|------|------|
| 切 | Cut | 直接切换，最常用 |
| 硬切 | Hard Cut | 无过渡的突然切换，制造冲击 |
| 软切 | Soft Cut | 稍有重叠的平滑切换 |
| 淡入 | Fade In | 从黑/白画面渐渐显示 |
| 淡出 | Fade Out | 画面渐渐消失到黑/白 |
| 溶解 | Dissolve / Cross-dissolve | 前一画面渐隐同时后一画面渐显 |
| 划像 | Wipe | 前一画面被后一画面推走 |
| 匹配剪辑 | Match Cut | 前后画面形状/动作相似，无缝衔接 |
| 跳切 | Jump Cut | 同一画面突然跳跃，时间流逝感 |
| L切 | L-Cut | 画面已切到下一场景，声音还在前一场景 |
| J切 | J-Cut | 下一场景的声音先出现，画面后切入 |

## 色调方案参考

| 情绪/场景 | 色调 | 典型应用 |
|-----------|------|----------|
| 温暖回忆 | 暖黄、金色、柔光 | 怀旧、家庭、幸福时光 |
| 冷酷现实 | 冷蓝、灰白、硬光 | 现代都市、疏离、真相 |
| 悬疑紧张 | 深蓝、暗绿、明暗对比 | 调查、背叛、阴谋 |
| 热情冲突 | 深红、橙色、高对比 | 对峙、愤怒、情欲 |
| 悲伤绝望 | 灰白、苍白、冷光 | 死亡、失败、离别 |
| 梦幻重生 | 柔白、淡金、光晕 | 重生、觉醒、希望 |
| 猥琐秘密 | 暗黄、昏暗、阴影 | 偷情、阴谋、地下交易 |
| 胜利高潮 | 金色、明亮、侧光 | 真相大白、正义伸张 |
